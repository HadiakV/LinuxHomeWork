1.Task
	1.Создать локальную папку, в которую будет скопирован репозиторий.

	2.Перейти на страницу репозитория на GitHub.com.

	3.Скопировать ссылку на репозиторий, нажав на зеленую кнопку "Code" и выбрав нужный вариант (HTTPS или SSH).

	3.Открыть командную строку или терминал на своем компьютере и перейти в папку, созданную на втором шаге.

	4.Ввести команду git clone и вставить скопированную ссылку на репозиторий, нажать Enter.
2.Task
	1.Инициализируйте локальный репозиторий в вашем каталоге, выполнив команду в командной строке:
	git init

	2.Добавьте все файлы в ваш локальный репозиторий:
	git add . (точкой показываем што хотим добаить все измененные файлы в текущей директории)

	3.Сделайте коммит изменений с сообщением:
	git commit -m "commit"

	4.Создайте новый репозиторий на GitHub и скопируйте URL-адрес репозитория.

	5.Свяжите ваш локальный репозиторий с удаленным репозиторием на GitHub:
	git remote add origin <укажите удаленный репозиторий точнее ссылку URL>

	6.Загрузите локальный репозиторий на GitHub:
	git push -u origin master
3.Task
	1.Открыть терминал (для Mac и Linux) или командную строку (для Windows).

	2.Перейти в каталог, где находится локальный репозиторий, используя команду cd.

	3.Выполнить команду git pull origin <name>, где <name> - это название ветки, которую вы хотите обновить на 
	  своем локальном компьютере. Например, если вы хотите обновить ветку litleUser, команда 
	  будет выглядеть так: git pull origin litleUser.

	Git сравнит версию на вашем локальном компьютере с версией на Github и загрузит изменения с удаленного 
	репозитория в ваш локальный репозиторий.
4.task
	'git status' - показывает текущее состояние рабочей директории, в том числе какие файлы были изменены,
	 добавлены или удалены, и готовы ли они к коммиту.

	'git log' - показывает историю коммитов в репозитории, включая информацию о коммитах, такую как хэш коммита, 
	 автор, дата и время, а также сообщение коммита.

	'git remote -v' - показывает список удаленных репозиториев, связанных с вашим локальным репозиторием, и 
	выводит URL для чтения и записи для каждого удаленного репозитория.
